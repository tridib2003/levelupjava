Enumeration

  • Enumeration is a group of predefined constants that defines a new data type.

  • The enum (enumeration) variable can only store one of those values that have been predefined for it.

  • Enums being constants, the values of an enum set are in uppercase letters.

  • Enums in Java defines a class type. Thus, an enumeration in Java can have constructors, methods, instance variables, and even implement interfaces.

  • An enumeration is created using the enum keyword. The listed values of an enum are called enumeration constants.


    • General syntax:

    public enum Planets{ MERCURY, VENUS, EARTH, MARS, JUPITER, SATURN, URANUS, NEPTUNE }

  • We can compare two enumeration constants using the == operator.

  • An enum value can be used to control the flow of a switch statement.

  • Java compiler internally creates a static and final class for an enum.

  • All enums contains some predefined methods like values() and valueOf(). The values() method returns an array containing all the enumeration constants. The valueOf() method returns the enumeration constant which matches the string parameter being passed to the method.
